# BRS Web Frontend

Frontend application for the Bank Reconciliation SaaS platform.

## Tech stack

- React 19 + TypeScript
- Vite
- React Router
- TanStack Query
- Tailwind CSS

## Scripts

- `npm run dev` - Start local development server.
- `npm run build` - Build production assets.
- `npm run preview` - Preview production build locally.
- `npm run lint` - Run ESLint checks.

## Configuration

- `VITE_API_URL`: API base URL used by the frontend.
- `VITE_API_PROXY_TARGET`: local dev proxy target for `/api` (defaults to `http://localhost:9001`).
